# Zvuk s řídícími prvky \(controls\)

`stimulus(controls)` zobrazí řídící prvky a tlačítka pro přehrání a změnu úseku

```text
test demo1zvuky
  screen Druhá obrazovka
  stimulus(controls) 1.wav
```
